## ✨ 功能特性

- 📋 **12道专业问卷** - 基于狗狗行为学设计的标准化量表
- 🎯 **6大核心维度** - 社交性、精力、自信、反应、可训练性、好奇心
- 🧠 **精准 MBTI 映射** - 12种狗狗性格类型，覆盖所有组合
- 💡 **详细性格分析** - 包含性格特点描述和相处建议
- 🎨 **简洁美观的界面** - 移动端友好，操作流畅

## 📖 使用方法

1. 阅读作答说明
2. 根据狗狗过去两周的典型行为作答
3. 每题从 A-E 选择最符合的选项
4. 完成后查看 MBTI 类型结果

### 评分标准
- **A 从不** = 0分
- **B 很少** = 1分
- **C 有时** = 2分
- **D 经常** = 3分
- **E 总是** = 4分

（反向题自动处理）

## 🐕 12种狗狗MBTI类型

### 外向型 (E)
| 类型 | 名称 | 描述 |
|------|------|------|
| ENTJ | 社交领队犬 | 自信、主导、反应快，常成为"狗群核心" |
| ENTP | 快乐冒险犬 | 精力充沛、好奇心强，喜欢变化和新刺激 |
| ESTJ | 模范犬 | 规则感强、听指令、稳定可靠，非常好训练 |
| ESFJ | 亲和陪伴犬 | 亲人、情绪稳定、喜欢互动，典型家庭犬 |
| ENFP | 热情探索犬 | 外向又感性，情绪表达强，黏人但爱玩 |
| ESFP | 开心果犬 | 活在当下，爱玩爱闹，行动先于思考 |

### 内向型 (I)
| 类型 | 名称 | 描述 |
|------|------|------|
| INTJ | 冷静规划犬 | 独立、观察力强，习惯先判断再行动 |
| INTP | 研究型观察犬 | 不黏人但好奇，常自己"研究世界" |
| ISTJ | 守序犬 | 稳定、依赖熟悉环境，非常忠诚 |
| ISFJ | 温柔照看犬 | 安静、情绪细腻，对主人依恋度高 |
| INFJ | 敏感洞察犬 | 慢热、感知力强，对环境变化非常敏感 |
| INFP | 温柔理想犬 | 情绪丰富、需要安全感，依恋关系深 |
